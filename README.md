# Twimodo

## これはなに？
一人のつぶやきメモ帳。  
だから、いいねとかリツイートとかありません。  
でも、投稿日時の部分が個別リンクになっているので、単品で共有はOK。

## なんで作ったの？
twitter...いやXに出すほどのものじゃない投稿を自分の中だけで消化したくて。



## 使い方
お気持ちを投稿する時は
```
npm start
```

- 一覧表示：http://localhost:3000
- 投稿画面：http://localhost:3000/post

一通りお気持ちを吐き出したらサーバーを止めて、
```
npm run generate
```

public配下にファイルが出来上がるのでrsyncとかなんなりでwebサーバーやS3にでも保存して。


他にもコマンドラインで投稿できたりするけど、それは[仕様書](仕様書.md)を読んでみて。
